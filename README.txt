這個檔案 `wave_animation.html` 是一個用於顯示動態波浪動畫的 HTML 檔案。其主要功能包括：

1. **波浪動畫**：
   - 使用 JavaScript 和 HTML5 Canvas 繪製動態波浪。
   - 波浪的波幅、頻率、速度和顏色是隨機生成的。
   - 滑鼠移動時，波浪的波幅會根據滑鼠的垂直位置動態調整。

2. **背景粒子效果**：
   - 在背景中加入了閃爍的粒子效果，增強視覺吸引力。
   - 粒子會隨機生成，並以不同的透明度閃爍。

3. **響應式設計**：
   - 當視窗大小改變時，Canvas 會自動調整大小，並重新初始化波浪和粒子。

此檔案適合用於展示動態視覺效果，並可作為學習 HTML5 Canvas 和動畫技術的範例。
